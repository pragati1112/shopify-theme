# Shopify Theme Customization

A custom Shopify theme project created and customized for learning and practice purposes. This project includes responsive layouts, reusable Shopify sections, Liquid templating, and modern UI customization.

## 🚀 Features

- Responsive Shopify theme
- Custom sections and schema settings
- Homepage customization
- Product and collection layouts
- Reusable components
- Mobile-friendly design
- Clean and organized code structure

## 🛠️ Technologies Used

- Shopify Liquid
- HTML5
- CSS3
- JavaScript
- Shopify Schema
- Git & GitHub

## 📂 Folder Structure

- `sections/` → Custom Shopify sections
- `snippets/` → Reusable code snippets
- `templates/` → Page templates
- `assets/` → CSS, JS, images, fonts
- `layout/` → Theme layout files

## 📌 Purpose

This project was built to practice:
- Shopify theme development
- Liquid templating
- Theme customization
- GitHub version control workflow
